## Structure

Each folder contains the agent's personal data files:
- `SOUL.md` — personality and core values
- `IDENTITY.md` — name, role, vibe
- `AGENTS.md` — workspace rules and memory conventions
- `MEMORY.md` — long-term curated memory
- `USER.md` — context about Lambert
- `TOOLS.md` — local tool notes
- `HEARTBEAT.md` — periodic task checklist

## Sync Rule

> ⚠️ **When Lambert asks you to update your personal data, skills, identity, soul, memory, or any agent-level config — after making the change locally, open a PR to this repo to keep it in sync.**

Never push directly to `master`. Always PR so Lambert can review.
