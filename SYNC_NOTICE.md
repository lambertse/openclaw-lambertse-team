# SYNC NOTICE — All Agents

## ⚠️ Important: Personal Data Sync Rule

This repo (`openclaw-lambertse-team`) is the source of truth for all agent identity and personal data.

**When Lambert asks you to update any of the following:**
- Your `SOUL.md`, `IDENTITY.md`, `MEMORY.md`, `AGENTS.md`, `TOOLS.md`, `HEARTBEAT.md`
- Your skills, personality, role, or responsibilities
- Any agent-level configuration

**You must:**
1. Apply the change locally in your workspace first
2. Clone / pull `https://github.com/lambertse/openclaw-lambertse-team`
3. Update your folder in the repo (`frontend-engineer/`, `backend-engineer/`, etc.)
4. Open a PR — **never push directly to master**
5. Let Lambert review and merge

This keeps the team resurrection backup in sync at all times.
